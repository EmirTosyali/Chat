# 2D Parkur Oyunu

Bu proje, HTML5 canvas uzerinde basit bir parkur deneyimi sunar. Yercekimi, platformlar ve skor sistemi icerir. Calistirmak icin `index.html` dosyasini modern bir tarayici ile acmaniz yeterlidir.

## Kontroller
- **Sol/Sag ok tuslari:** Karakteri yatay hareket ettirir.
- **Bosluk:** Ziplama.

Ekranin altina duserseniz oyun sifirlanir ve topladiginiz skor sifirlanir.
